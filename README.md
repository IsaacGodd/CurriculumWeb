## ✨ Features
- **🌐 Bilingual UI:** Full English and Spanish support using `react-i18next`.
- **📱 Fully Responsive:** Mobile-first design adopting glassmorphism and modern CSS practices.
- **🛡️ Cyber & Web Split:** Projects are rigorously categorized under `Web Development` and `Cybersecurity` sections.
- **📄 Instant CV Downloads:** Serve highly specific resumes depending on the recruiter's domain.
- **💡 Project Spotlights:** Deep dive into specific projects with embedded architectures, network topologies, and video demos.

## 🛠️ Tech Stack
- **Frameworks:** React, Vite
- **Styling:** Tailwind CSS, Custom Utilities (`index.css`)
- **Routing & State:** React Router DOM, React Hooks 
- **Animations:** Framer Motion
- **Tooling:** ESLint, PostCSS

## 🚀 Run Locally

If you'd like to check out the code and run it in your local environment, follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/IsaacGodd/CurriculumWeb.git
   cd CurriculumWeb/cv-app
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   Agrega `--host` si deseas exponerlo a tu red Wi-Fi local.
   ```bash
   npm run dev
   ```
